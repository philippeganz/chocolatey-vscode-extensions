# Security Policy

## Supported Versions

Since this repository serves as an Automated Updater (AU) factory for Chocolatey packages, we strictly support the latest available version of the repository templates and scripts.

For the VS Code extensions themselves, security support and patches are governed by the upstream extension publishers on the Visual Studio Marketplace.

| Component | Supported |
| ------- | ------------------ |
| Latest Factory Scripts | :white_check_mark: |
| VS Code Extensions | Handled by upstream publishers |

## Reporting a Vulnerability

If you discover a security vulnerability within the packaging scripts or the AU automation process, please **do not open a public issue**.

Instead, please report it privately using **GitHub's Private Vulnerability Reporting**:

1. Go to the **Security** tab of this repository.
2. Click **Report a vulnerability** to privately notify the maintainers.

Please provide a detailed description of the issue, steps to reproduce, and any relevant logs. We aim to acknowledge receipt of your report within 48 hours.
